Spring Boot Web App

This simple CRUD application was created using SpringBoot, Bootstrap, Spring Security, Thymeleaf, contains database connection to PostgreSQL by Anhelina Kulakova as a Final project for Server Programming course. The purpose of this application is to create a database of the job applications made by a user in order to keep track of actual vacancies meant for applying in future and also jobs which were already applied as well as the status of these applications. The app contains two demo users: user/user (role=USER) and admin/admin (role=ADMIN). The idea of the app originated from my friends who are saving such information using Exel spreadsheets.
